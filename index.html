Лаборатория перспективных систем управления при Физтехе занимается проектированием и разработкой полного стека технологий для управления разнородными беспилотными аппаратами как по отдельности, так и в составе групп.

Большая часть разрабатываемых в рамках лаборатории технологий и продуктов доступна бесплатно в открытом виде под свободными лицензиями на этом сайте.

Разрабатываемые продукты и технологии

Приложения

  • Мобильный пункт управления dev — приложение для управления беспилотными аппаратами работающими по протоколам Mavlink и борт-земля. Используемые технологии: C++, Qt.
  • ГеоТаргет dev — приложение для управления беспилотными аппаратами работающими по протоколам Mavlink и борт-земля с модулем картографии в 3D. Используемые технологии: Java, Scala, JavaFX, Akka, NASA WorldWind.

Библиотеки и средства разработки

  • BMCL dev — бортовая стандартная библиотека для языка C++
  • Библиотека для языка C++ реализующая работу по протоколу борт-земля для бортовых систем будет доступна позже
  • Предметная модель описания бортовых систем Decode α, включает в себя следующие модули:
    • Рантайм для работы с моделью и средства разработки моделей бортовых компонентов для языка Scala
    • Плагин для JetBrains IDEA позволяющий производить редактирование исходного кода на языке Decode
    • Генератор документации в формате HTML для компонентов описанных на Decode
    • Генератор бортового кода аппарата для языка C
    • Генератор описания компонентов на языке Decode на основе описания обменного протокола Mavlink

Протоколы и стандарты

  • Протокольный фреймворк PPF α
  • Описание предметно-ориентированного языка Decode α
  • Протокол организации взаимодействия между аппаратами Photon α
  • Документация по бортовым компонентам протокола Photon dev
