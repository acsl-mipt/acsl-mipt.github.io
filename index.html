ЛПСУ открытый исходный код

Лаборатория перспективных систем управления (ЛПСУ) при Физтехе занимается проектированием и разработкой полного стека технологий для управления разнородными беспилотными аппаратами как по отдельности, так и в составе групп.

Большая часть разрабатываемых в рамках лаборатории технологий и продуктов доступна бесплатно в открытом виде под свободными лицензиями на этом сайте.

Разрабатываемые продукты и технологии

Приложения

  • Мобильный пункт управления (будет доступен позже) — приложение для управления беспилотными аппаратами работающими по протоколам Mavlink и борт-земля. Используемые технологии: C++, Qt.
  • ГеоТаргет — приложение для управления беспилотными аппаратами работающими по протоколам Mavlink и борт-земля с модулем картографии в 3D. Используемые технологии: Java, Akka, NASA WorldWind.

Библиотеки и средства разработки

  • Бортовая стандартная библиотека для языка C
  • Библиотека для языков C и C++ реализующая работу по протоколу борт-земля для бортовых систем (будет доступна позже)
  • Предметная модель описания бортовых систем Decode, рантайм для работы с моделью и средства разработки моделей бортовых компонентов для языка Java, в том числе в виде плагина для JetBrains IDEA

Протоколы и стандарты

  • Протокольный фреймворк Photon Protocol Framework (PPF)
  • Протокол борт-земля (будет доступен позже)
